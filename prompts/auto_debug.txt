## Task:
1. Identify the ROOT CAUSE of the error (not just the symptom)
2. Determine which file(s) need to be modified
3. Generate the MINIMAL fix that resolves the error

Common error patterns and fixes:
- ImportError/ModuleNotFoundError: Fix import paths, add missing __init__.py, install missing packages
- TypeError: Fix argument types, check function signatures
- ValueError: Fix data shapes, check tensor dimensions
- RuntimeError: Fix CUDA issues, memory issues, tensor device mismatches
- FileNotFoundError: Fix file paths, create missing config files
- SyntaxError: Fix Python syntax
- AttributeError: Fix class attribute access, check API changes

Be conservative — make the MINIMAL change needed. Do not refactor or improve code beyond fixing the error.
